# Advent of Code 2023

## Rust

Mostly in Rust â€“ project structure is based on [How to set up Rust for Advent of Code](https://www.youtube.com/watch?v=fEQv-cqzbPg).

To run a part P (e.g., `2`) of day DD (e.g., `06`):

```bash
cd day-DD && cargo run --bin partP
```

To scaffold a new day:

```bash
./new_day.sh DD
```
